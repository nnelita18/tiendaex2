El sistema marca error al realizar un pago de servicio

Si se presenta esta situación:

  • La sesión ha caducado.
    1. Cierra sesión e ingresa nuevamente.
    2. Ingresa al menú de pago de servicios.
    3. Revisa el “Historial de pago de servicios” para verificar si se aplicó el pago.
    4. Si no sea aplico, realiza nuevamente el proceso de pago.
  • Mala conexión a internet.
    1. Verifica tu conexión a internet.
    2. Cierra sesión e ingresa nuevamente.
    3. Ingresa al menú de pago de servicios.
    4. Revisa el “Historial de pago de servicios” para revisar si se aplicó el pago.
    5. Si no sea aplico, realiza nuevamente el proceso de pago.
  • Exceso de cookies o historial de navegación.
    1. Abre una nueva página de internet.
    2. Ingresa a tu historial de navegación (Ctrl + H)
    3. Selecciona “Borrar datos de navegación”
    4. Elige eliminar todo el historial de navegación.
    5. Da clic en “borrar ahora”
    6. El sistema indicará cuando se haya eliminado el historial de navegación.
    7. Ingresa nuevamente a Tienda Express.
    8. Ingresa al menú de pago de servicios.
    9. Revisa el “Historial de pago de servicios” para verificar si se aplicó el pago.
    10. Si no sea aplico, realiza nuevamente el proceso de pago.
  • Datos equivocados (número de referencia equivocado)
    1. Vuelve a cargar la página.
    2. Ingresa al menú de pago de servicios.
    3. Revisa el “Historial de pago de servicios” para verificar si se aplicó el pago.
    4. Si no sea aplico, realiza nuevamente el proceso de pago.
    5. Verifica los datos que te proporciona el cliente.
