El sistema marca error al realizar una recarga

Si el sistema marca error, se queda procesando la recarga o se trabo:

  • La sesión ha caducado.
    1. Cierra sesión e ingresa nuevamente.
    2. Ingresa al menú de recargas.
    3. Revisa el "Historial de ultimas recargas" para verificar si se aplicó la recarga.
    4. Si no sea aplico, realiza nuevamente el proceso de recarga.
  • Mala conexión a internet.
    1. Verifica tu conexión a internet.
    2. Cierra sesión e ingresa nuevamente.
    3. Ingresa al menú de recargas.
    4. Revisa el "Historial de ultimas recargas para verificar si se aplicó la recarga.
    5. Si no sea aplico, realiza nuevamente el proceso de recarga.
  • Exceso de cookies o historial de navegación.
    1. Abre una nueva página de internet.
    2. Ingresa a tu historial de navegación (Ctrl + H)
    3. Selecciona "Borrar datos de navegación
    4. Elige eliminar todo el historial de navegación.
    5. Da clic en "borrar ahora
    6. El sistema indicará cuando se haya eliminado el historial de navegación.
    7. Ingresa nuevamente a Tienda Express.
    8. Ingresa al menú de recargas.
    9. Revisa el "Historial de ultimas recargas para verificar si se aplicó la recarga.
    10. Si no sea aplico, realiza nuevamente el proceso de recarga.
  • Datos equivocados (el número no pertenece a esa compañía o está mal escrito)
    1. Vuelve a cargar la página.
    2. Ingresa al menú de recargas.
    3. Revisa el "Historial de ultimas recargas para revisar si se aplicó la recarga.
    4. Si no sea aplico, realiza nuevamente el proceso de recarga
    5. Verifica los datos que te proporciona el cliente.
